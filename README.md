¿Qué es este proyecto?
Este sistema permite a compañías de seguros, talleres y peritos gestionar digitalmente todo el proceso de partes de accidentes, eliminando el papeleo físico y reduciendo los tiempos de gestión de 72 horas a menos de 2 horas.

Características Principales
-Gestión Completa de Partes
-Creación digital de partes de accidente
-Validación en tiempo real de datos
-Gestión de múltiples vehículos involucrados
-Asignación automática a peritos y talleres

Interfaz Moderna y Responsiva
-Diseño adaptado para móvil, tablet y escritorio
-Formularios inteligentes con progreso visual
-Panel de administración completo
-Exportación de datos en múltiples formatos

Seguridad y Compliance
-Autenticación de usuarios segura
-Cifrado de datos sensibles
-Registro de auditoría completo
-Cumplimiento RGPD/LOPD
